# Pete McPherson — Claude Code Plugins

Open source Claude Code plugins by [@petemcpherson](https://github.com/petemcpherson).

## Add this marketplace

```
/plugin marketplace add petemcpherson/claude-plugins
```

## Available plugins

### pete-loop
> Autonomous build loop for Claude Code

Set up a structured **spec → plan → loop** workflow that implements one task per fresh context window. Built for Claude Pro subscription users.

**Skills:**
- `/pete-loop:setup` — Scaffold the `pete/` folder in any project (run once per project)
- `/pete-loop:spec` — Guided AI session to build your `pete/spec.md`
- `/pete-loop:plan` — Generate a phased `pete/plan.md` from your spec

**Install:**
```
/plugin install pete-loop@petemcpherson-claude-plugins
```

[→ View pete-loop repo](https://github.com/petemcpherson/pete-loop)
